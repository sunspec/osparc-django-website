Key Performance Indicators

Key performance indicators (KPI) are presented for the set of plants defined in the report. The KPIs shown on the oSPARC dashboard apply to all oSPARC plants.

The sample interval used in all calculations is monthly; each plant's contributing value is the average over its values over the lifetime defined in the report - all time, for the dashboard report.

For each KPI, minimum, maximum, mean and median values are shown. These apply across the plants in the report; i.e. the minimum value is that of the plant with the lowest value for the KPI, the mean is the average across all plants, etc.

Three KPIs are reported: Yield, Performance Ratio and Storage State of Health. These are defined below.

The other values in the report are the measured values used to calculate the KPIs.

Yield

The standard Yield metric is considered to be the “bottom-line” indication of how well a system is performing.

Yield is calculated as follows:

Yield = monthly energy producion / Plant DC Rating

For each plant, monthly energy production is the average across all periods in the report. Mean, median, minimum and maxim values across the plant sample set are shown (see KPIs above).

Performance Ratio

Performance Ratio (PR) normalizes Yield for insolation, as follows:

PR = Yield / (POA Insolation/1000)

Storage State of Health

Storage state of health shows the remaining effectiveness of on-plant storage; it is the ratio of the current storage capacity divided by the initial (nameplate) storage capacity, as follows:

SoH = current capacity / nameplate capacity

Only two data points are used for each plant: original and current; state of health is independent of system age.
